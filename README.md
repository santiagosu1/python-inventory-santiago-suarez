# Inventory Management System

## Short Description
This project is a simple Python inventory management system. It allows users to add, view, update, and remove items from inventory. The program uses dictionaries, lists, tuples, sets, functions, classes, inheritance, loops, conditionals, and file handling.

## How to Run the Program
1. Make sure Python is installed on your computer.
2. Open the terminal in the project folder.
3. Run the following command:

```bash
python santiago_coursework1.py
